Typography& Iconography: Resume Build

This course taught me the importance of typographic and iconographic elements and how from a design perspective, elements such as space, contrast, weight (line and typeface weight) can allow or restrict a viewer from visualizing your content with ease.For this project I was challenged to learn about my style and identity. Design Essentials: US FOODS Brand Advertisment Set

As my second course on Photoshop, but my first hands-on experience with other Adobe applications like Illustrator and InDesign, I learned how to be a competent digital designer. The course also helped me make some of the icons you see on this site.In this project I was challenged to use what skills I had obtained to work within a brands guidelines (with specifications) to build a suite of advertisments and a cell sheet. This experience taught me that as a designer there will be times where your concepts cannot come to fruition if there a guidelines in place and following them isn't so bad. It can actually releive some pressure if you know what is expected of your work.

Web Fundamentals(html and css): NCT Fan Site

This class explained the very backbone of web programming, responsive web design as well as responsible web design (accessibility, organization and file compression). Originally, I admit, I struggled to create webpages that met these standards however after taking the Foundations in Web Design and Strategy course my understanding of the languages has improved dramatically. I learned that when it comes to programming languages, it is important to fully understand an elements capabilities and warnings on how to best utilize them.
